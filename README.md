## 🎮 Особенности проекта

- 🕹️ **Физика полета:** Аркадная модель управления с инерцией, сопротивлением воздуха и наклоном корпуса.
- 🌡️ **Система тепловизора:** Индикатор сигнала усиливается по мере приближения к источнику тепла.
- 🧭 **Навигация:** Динамическая красная стрелка-компас, указывающая направление к ближайшему пострадавшему.
- 📦 **Механика спасения:** Удерживайте прицел на цели в течение 2 секунд для сброса груза (с анимацией троса и контейнера).
- 💥 **Система повреждений:** Столкновения с границами карты или землей на высокой скорости приводят к крушению дрона.
- 📱 **Кроссплатформенность:** Полная поддержка мобильных устройств с экранным джойстиком и кнопками управления.
- ⚙️ **Адаптивная оптимизация:** Автоматическое определение слабых устройств (снижение полигональности, отключение теней и частиц).
- 🎵 **Процедурный звук:** Генерация звуковых эффектов (двигатель, тепловизор, спасение, крушение) через Web Audio API без внешних файлов.
- ♿ **Доступность:** Поддержка ARIA-атрибутов для скринридеров.

## 🕹️ Управление

### 💻 Десктоп (Клавиатура + Мышь)
| Клавиша | Действие |
| :--- | :--- |
| `WASD` или `Стрелки` | Горизонтальное движение |
| `Q` или `Shift` | Набор высоты |
| `E` или `Ctrl` | Снижение |
| `ЛКМ` (удерживать) | Сброс спасательного груза |
| `ESC` | Пауза / Меню |

### 📱 Мобильные устройства
- **Левый нижний угол:** Виртуальный джойстик для движения.
- **Правый нижний угол:** Кнопки высоты (`⬆` / `⬇`) и сброса груза (`📦`).
- **Правый верхний угол:** Кнопка паузы (`⏸`).

## 🚀 Как запустить

### Вариант 1: Локальный запуск (Рекомендуется)
Для корректной загрузки текстур и аудиофайлов лучше использовать локальный сервер:
1. Установите [Node.js](https://nodejs.org/) или [Python](https://www.python.org/).
2. Откройте терминал в папке с проектом.
3. Запустите сервер:
   - Python: `python -m http.server 8000`
   - Node.js (через npx): `npx serve`
   - VS Code: используйте расширение **Live Server**.
4. Откройте в браузере: `http://localhost:8000`

### Вариант 2: Простой запуск
Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge). *Примечание: некоторые браузеры могут блокировать загрузку локальных аудиофайлов из-за политик безопасности CORS.*

## 🛠️ Технические детали

- **Графический движок:** [Three.js](https://threejs.org/) (r128)
- **Архитектура:** Vanilla JavaScript (ES6+), без тяжелых фреймворков.
- **Оптимизация:** Использование Object Pooling для частиц и эффектов, чтобы избежать сборщика мусора (Garbage Collection) во время игры.
- **Хранение данных:** `localStorage` для записи рекордов.
- **Стили:** CSS Variables для динамической смены цветовой темы в зависимости от сложности.
